# Stage 2 — Trend & Reference Research

**Goal:** ground the work in reality and in the **current best-recognized** trends,
animations, and effects for this mode + subject. This stage is mandatory every run.

## Do

1. Read `reference-brief.md`, `intake.md`, and the mode playbook. Spawn a `researcher`.
2. Use web + image search to gather, for THIS mode and subject:
   - **Trends / animations / effects** — the current best-recognized styles,
     transitions, motion, and effects (start from the playbook's seeded 2026 list,
     then refresh live; name exemplars).
   - **Real-world reference** — how the subject/material actually looks; comparable
     work (films, ads, creators, sites) worth honoring or subverting; lighting + grade.
   - **What's achievable prompt-first** in Flow/Veo (and Nano Banana / Midjourney for
     stills), and what must be flagged as post (real logo/text, regulated claims).
3. Distill — don't dump links. Write `research.md`: grouped, concrete, usable
   observations with sources, plus a shortlist of the trend techniques to apply.

## Output

`research.md` — the trend + reference dossier that feeds the creative direction.
