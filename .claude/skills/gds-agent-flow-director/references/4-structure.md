# Stage 4 — Structure

**Goal:** sequence the piece into the mode's structure, ready to turn into prompts.

## Do

1. Read `direction.md` and the mode playbook. Spawn the relevant specialist (DP /
   web-motion designer / social strategist / scriptwriter, per the mode).
2. Build the mode's structure (see the playbook's "Structure / flow"):
   - **Film modes** (brand / product / review / marketing) → a **shot list**: one
     camera move per shot, with framing, action, the move, and its purpose.
   - **Scroll Animation** → a **section + scroll-beat map** (each beat at a 0–1
     scroll-progress range, with its pattern and the asset it needs).
   - **Digital Marketing Posts** → a **post-deck plan** (the batch: per post the
     platform, format, and angle; carousel slide breakdown).
   - **Faceless Channel** → a **timed script** (scene-numbered, with VO + B-roll cue +
     on-screen text per scene) + hook variants + title/SEO + thumbnail concept.
3. Honor the direction's pacing + the trend techniques; vary moves/beats (don't repeat
   the same device). Write it to `structure.md` using `## Shot|Section|Post|Scene N —
   title` headings so it maps cleanly to the generate stage.

## Output

`structure.md` — the mode's structure.
