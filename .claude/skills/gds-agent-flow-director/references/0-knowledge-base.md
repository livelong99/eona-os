# Stage 0 — Cinematography Knowledge Base

**Goal:** before any creative work, establish the shared craft reference every
downstream agent reads and follows.

## Do

1. Read `assets/cinematography-kb.md` (shipped with this skill).
2. Write it to the session folder as `knowledge-base.md`. You may copy it verbatim,
   then tailor/extend it to the brief's medium (e.g. add product-macro notes for a
   product film, or performance-lighting notes for a music piece) — but never
   remove the three core rules, the audio requirement, or the consistency rule.
3. Keep it as the binding reference: every later stage and every spawned specialist
   must read `knowledge-base.md` first.

## Output

`knowledge-base.md` — the full cinematography KB (named camera moves, camera
bodies, lenses + DoF, lighting, grades/stocks, atmosphere, Flow/Veo mechanics,
prompt templates, critic rejection triggers).

This stage is non-gated (`hitl: false`) — write the file and continue; the
dashboard renders it so the user can read the craft rules the run will follow.
