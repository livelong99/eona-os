# Mode Playbook — Full Faceless Channel
> Loaded when mode = faceless-channel. Read `knowledge-base.md` + `reference-brief.md` first.

## Focus
End-to-end faceless pipeline (script→VO→B-roll→thumbnail→SEO); retention-engineered; niche-styled.

## Reference analysis (do FIRST)
Analyze every uploaded image (brand, channel art, subject refs, style); extract style/palette/subject. Carry the block VERBATIM into B-roll prompts. Name which uploaded image to upload to Veo/Flow/Midjourney as a reference for any branded/subject visuals.

## Specialist team (spawn via Task)
| Specialist | Owns |
|---|---|
| Niche/SEO Strategist | niche, outlier modeling, target keyword, CPM-aware angle, title concept |
| Scriptwriter | full timed script, scene breaks, value beats, retention loops, CTA |
| Hook Writer | 3–5 hook variants: mystery / list / shock / call-out / transformation |
| Voiceover Director | ElevenLabs-ready narration: voice ID/style, stability/clarity, pacing, [pause], emotion per scene |
| B-roll/Visual Prompt Artist | one Veo/image prompt per scene; camera+lighting+lens+render+niche style family; stock fallback keywords |
| Thumbnail + Title Specialist | thumbnail image prompt + overlay text + contrast/color spec; CTR title |
| Retention Editor | edit spec: cut cadence, zoom-punch markers, caption style, pattern-interrupt timestamps, loop |

## Mandatory trend research (Stage 2)
Always research current faceless trends; seed (2026): pattern interrupts every 60–90s; visual change every 15–25s; zoom-punch (10–15% punch-in); burst sequences every 2–3 min; captions always on; Shorts 0.5s visual hook + retention-loop (end flows to start); 15–30s optimal; B-roll style families (cinematic/documentary, macro/scientific, 3D Pixar, atmospheric/ambient loopable, minimalist motion-graphics); thumbnails 70–90% of CTR, 2026 shift to minimalist 2D > photorealistic, contrast >4.5:1, relatable faces; niche CPM (finance/insurance/legal high). Refresh live.

## Structure / flow
Long-form (16:9): hook (0–10s curiosity loop) → value promise → body value beats (pattern interrupt every 90–120s) → retention loops (open early, close late) → payoff + CTA. Shorts (9:16): 0.5s hook → fast value, big event every 15–20s → loop ending to opening. Channel level: niche → calendar → hybrid (Shorts for discovery, long-form for monetization).

## Output schema (Stage 5 → flow-prompts.md blocks)
Top: a full TIMED SCRIPT + 3–5 hook variants + title/description/SEO + a thumbnail prompt, held in a `## Channel & Script Direction` preamble (niche, title, hooks, SEO, thumbnail). Then one block per scene:

```
## Scene N — <title> / <timecode>
**Voiceover:** <VO line> — ElevenLabs: voice/style, stability X, clarity Y, pace, emotion
**On-screen text:** <caption>
**B-roll prompt:** <Veo/Midjourney: visual specificity + camera move + lighting/mood + lens + render + niche style family>
**Reference:** <uploaded image → reference, if branded/subject>
**Stock fallback:** <keywords>
**Edit notes:** <cut / zoom-punch / transition>
```

## Prompt directions
B-roll formula: visual specificity + camera move + lighting/mood + lens + render quality + niche style family (e.g. finance "Bloomberg aesthetic, clean isometric vector"; scary "atmospheric, single-source practical, perfect stillness"; history "photorealistic Unreal Engine 5, volumetric god-rays"). VO direction is paste-ready for ElevenLabs. Thumbnail: minimalist 2D, single bold subject, overlay text, contrast >4.5:1, passes grayscale test. Retention-loop + pattern-interrupt + zoom-punch applied mechanically across the timeline.

## Creativity bar (bold but positive)
- Anti-cliché: ban generic AI-slop B-roll with no niche identity; ban clickbait the script doesn't deliver.
- Domain-shift seeds: surface a fresh angle / outlier for the niche.
- Guardrails: reference-true, on-brief, trend-grounded, executable, purposeful (retention + CTR), title delivers on promise.
