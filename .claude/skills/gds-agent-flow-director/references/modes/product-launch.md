# Mode Playbook — Product Launch
> Loaded when mode = product-launch. Read `knowledge-base.md` + `reference-brief.md` first.

## Focus
Hero product film; macro→micro proof, exploded/levitation, light-rake; resolve on a locked hero shot.

## Reference analysis (do FIRST)
Analyze every uploaded product image; extract form/geometry, finish/material per surface, color hexes, logo placement, distinguishing details. Carry the reference-brief product block VERBATIM. Every block names which uploaded image to upload to Flow as Ingredient; build a Nano Banana HERO STILL seeded from the upload, then animate it (reference-to-video, fixed seed).

## Specialist team (spawn via Task)
| Specialist | Brings to the table |
|---|---|
| Product Strategist / Creative Director | One message, one hero feature, one CTA, and which trend beat each shot serves |
| Industrial / Visual Designer | The hero still — form, per-surface material callouts, visual DNA, logo placement |
| DP | Macro/anamorphic lens, dolly/orbit moves, weighted camera inertia, light-rake, DoF |
| Motion / FX Designer | Exploded view, levitation, liquid sim, seamless loop (first frame = last frame) |
| Sound Designer | SFX tied to visible action, score swell on the drop, satisfying click/shimmer |

## Mandatory trend research (Stage 2)
Always research current best product-launch trends/effects; seed (2026): macro-to-micro push in first 4s; ALWAYS resolve to a locked realistic hero shot; exploded/teardown + x-ray; magnetic levitation / zero-shake glides / liquid-dynamics; animated light raking on metals + soft rim on polymers; isolate 3+ distinct material textures; weighted camera physics (mass) vs frictionless glide (lightness); kinetic internals; 360 orbits + satisfying seamless loops. Refresh live.

## Structure / flow
build-up/tease (extreme macro, dark studio, curiosity) → drop/hero reveal (macro→micro resolves to locked hero, light rake) → feature beats (exploded/levitation/liquid, one feature + one material each) → in-context use → CTA (specific, on-screen ≥5s). 15–30s social (9:16) / 60–90s landing / 16:9 YouTube; chain 4/6/8s clips.

## Output schema (Stage 5 → flow-prompts.md blocks)
Emit a `## Look Bible` preamble (studio, lens family, material vocabulary, fixed seed, locked-hero description) followed by a hero-still prompt, then per shot:

```
## Shot N — title
**Image prompt:** <Nano Banana hero-still or frame, seeded from upload>
**Video prompt:** <motion + inline AUDIO: SFX/score cues>
**Settings:** Aspect · Resolution · Seed · Flow feature
**Reference:** <uploaded image → Flow Ingredient + feature>
**Negative:** <positive-form negatives>
**Consistency:** <reuse identical reference + fixed seed; reference the locked still>
```
Reuse identical reference + fixed seed across all shots so the product stays reference-true.

## Prompt directions
Macro/anamorphic lens language, studio lighting, describe EACH finish separately (3+ materials — e.g. brushed-aluminum chassis, matte-polymer collar, polished-glass face); weighted vs frictionless camera; light that travels across the logo; reference the locked still explicitly in every shot; timestamp multi-shot blocks to chain a beat sequence; restrained premium audio tied to visible actions (3–5 ambient max); write negatives as positives (state what you want, not what you fear).

## Creativity bar (bold but positive)
- Anti-cliché: ban generic spinning-product-on-turntable-with-sparkles done lazily; ban physics-defying middle that never resolves.
- Domain-shift seeds: map the product's core benefit onto a distant domain (e.g. "battery life" → a desert that never runs dry; "precision" → a watchmaker's loupe).
- Guardrails: reference-true (exact product form/material/color), on-brief, trend-grounded, Flow-executable, purposeful (every beat proves a feature).
