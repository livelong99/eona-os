# Stage 3 — Creative Direction (divergent → convergent)

**Goal:** produce ONE binding creative direction for this mode — bold but on-target —
that governs everything downstream.

## Do

1. Read `reference-brief.md`, `research.md`, and the mode playbook. Spawn the mode's
   creative team via `Task` (see the playbook), plus the `critic`.
2. **Divergent:** generate **3–5 genuinely distinct concept directions**, each from a
   different creative lens (e.g. minimalist / surrealist / documentary-realist /
   maximalist). Apply the anti-cliché bar (ban the obvious metaphor) and domain-shift
   (map the subject's core tension onto a distant domain). Novelty = a fresh remix of
   the researched trends, never an unproven gimmick.
3. **Convergent:** the `critic` + you score each concept against the **five guardrails**
   — reference-true, on-brief/on-brand, trend-grounded, Flow-executable, purposeful —
   and **synthesize the winner**, grafting the best of the runners-up. The critic
   demands bolder/more distinctive where safe, and kills cliché + reference-drift.
4. Write `direction.md` — the binding direction in the mode's terms (Look Bible /
   Motion Direction / Content Direction / Channel & Script Direction). Include the
   **identity/product block** (verbatim, from `reference-brief.md`) to repeat in every
   prompt, the palette/grade/lens/lighting (or motion/format) language, pacing, and
   the chosen aspect ratio(s).

## Output

`direction.md` — the binding creative identity. Every later prompt conforms to it.

Gate (`hitl: true`).
