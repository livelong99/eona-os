# Screenshots

Drop dashboard captures here, then uncomment the screenshot block in the root [`README.md`](../../README.md#-screenshots).

## Shot list (suggested)

Run the stack (`scripts/install.sh`, then open <http://127.0.0.1:3737>) and capture:

| File | Screen | Notes |
|------|--------|-------|
| `home.png` | Home | The Aurora Orb landing — your hero image |
| `workspace.png` | Workspace | Live run lanes mid-execution; the git panel open |
| `memory.png` | Memory | The 3D knowledge globe |
| `brainstorm.png` | Brainstorm | An ideation swarm in progress |
| `hero.gif` | Workspace | **Highest impact:** a 5–10s loop of a live run |

## Tips

- Capture at 2× / Retina, then keep file sizes reasonable (< ~1 MB for PNGs, < ~5 MB for GIFs).
- Use a clean window without personal data in the vault.
- For the GIF, [Kap](https://getkap.co/) (macOS) or [Peek](https://github.com/phw/peek) (Linux) work well.
- Consistent window size across shots looks more polished in the README grid.
