# Agent OS — Deck (Dock) Icon Prompt Suite · APP-ICON QUALITY · LIGHT

**Target model:** Nano Banana Pro (Gemini 3 Pro Image) · generate in **Google Flow**
**Format:** 1:1 square · **light background, glyph fills ~90–95%** · output at **4K**
**Updated:** 2026-06-24

Eight **high-quality, modern app icons** — one per dock page — at the polish level of **Chrome, Safari,
Docker** and the **Home aurora orb** (the approved reference). Each glyph is a rich, dimensional, glossy
mark with smooth gradients, soft volumetric shading, crisp highlights and fine detail — sitting **large
(~90–95%)** on its own **light pastel background** that fills the whole square. Premium and characterful,
**not** flat line-art and **not** a crude drawing.

> [!IMPORTANT] Quality bar (this is the whole point)
> - **App-icon grade.** Think Chrome / Safari / Docker / the Home orb: **smooth rich color gradients,
>   soft dimensional volume, subtle gloss and clean highlights, gentle ambient shading, crisp fine
>   detail.** Vibrant, deep, polished. Lots of attention to detail.
> - **Dimensional, not flat.** The mark has real form and depth like the orb — but it is a clean modern
>   icon, **not** heavy 2010s skeuomorphism, **not** a glass slab/tile.
> - **No container.** No border, no outline frame, no tile, no squircle around the icon.
> - **Light background, full bleed.** A soft light pastel color fills the entire square to every edge.
> - **Big glyph**, ~90–95% of the canvas, thin even margin. Square 1:1. Cohesive family.

## How to run (consistency workflow)

1. **Generate `01 · Home` first** — the aurora orb is the family key-frame; it sets the gloss, gradient
   richness, shading and detail level every other icon must match.
2. For `02–08`, upload `01 · Home` in Flow and append: *"Use Image 1 as a strict STYLE reference —
   match its dimensional app-icon finish: smooth rich gradients, soft glossy volume, clean highlights,
   gentle shading and crisp detail level, on a light pastel background, glyph ~90–95%. Only the subject
   and its background tint change. No tile, no border, no frame, no heavy skeuomorphism."*
3. **Pin one seed** across all eight (e.g. `--seed=4242`).
4. Revise via `/edit` on the saved PNG, naming only the delta ("keep everything else identical").
5. Verify: Chrome/Safari/Docker-grade polish, dimensional and detailed (not flat/drawn), light
   background to all edges, no border/frame/tile, one centered glyph filling ~90–95%.

## Notes
- **Match the orb.** The aurora orb is the quality bar: smooth rich gradients, soft glossy volume, clean
  highlights, fine detail. Every other icon should feel like it belongs in the same dock as Chrome,
  Safari and Docker — dimensional and polished, never flat or drawn.

  no border or frame; background and corner shaping happen in code later.
- **Light pastel background per page**, glyph ~90–95% — bright, colorful, one cohesive family via shared
  finish and palette.
- 4K output for crisp HD; downscale to 1024/512/256/128 for the dock.
- Files carry an invisible SynthID watermark + C2PA credentials (Nano Banana Pro).
- After generation, drop the eight PNGs into `dashboard/public/icons/` (same filenames) to swap the dock.
