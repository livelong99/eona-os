# Agent OS — Deck (Dock) Icon Prompt Suite

**Target model:** Nano Banana Pro (Gemini 3 Pro Image) · generate in **Google Flow**
**Format:** 1:1 square · **full-bleed icon (fills the frame edge-to-edge)** · output at **4K** · transparent only in the four corner cut-outs
**Updated:** 2026-06-19

Eight high-definition icons — **one per dock page** — in a photoreal macOS "Liquid Glass" squircle
finish. **Full-bleed:** the squircle tile *is* the entire image; it extends to all four canvas edges
with no margin, no padding, no surrounding scene. One locked glass treatment binds the family; each
glyph is domain-shifted off its cliché and sized large.

---

## How to run (consistency workflow)

1. **Generate `01 · Home` first** — it's the family key-frame.
2. For `02–08`, upload `01 · Home` in Flow and append: *"Use Image 1 as a strict STYLE reference —
   match its full-bleed squircle, corner radius, glass thickness, layer depth, inner glow, top
   specular highlight, edge bevel and finish exactly; only the central glyph and its colorway change."*
3. **Pin one seed** across all eight (e.g. `--seed=4242`).
4. Revise via `/edit` on the saved PNG, naming only the delta ("keep everything else identical").
5. Verify: tile reaches all four edges, no outer margin/background, no text/letters, glyph ~70% of tile.

## Notes
- **Full-bleed** = the squircle is the whole image (touches all four edges); only the rounded corner
  cut-outs are transparent. This matches how macOS renders dock icons and avoids dead canvas margin.
- Shared dark graphite-glass base; only the under-glass **bloom** + glyph colorway shift per page →
  one cohesive family, eight instantly distinct marks.
- 4K output for crisp HD; downscale to 1024/512/256/128 for the dock.
- All outputs carry an invisible SynthID watermark + C2PA credentials (Nano Banana Pro).
- If you instead want the glyph to **bleed past the squircle edge** for a more dramatic, modern look,
  append: *"the glyph extends slightly beyond the tile's rounded edge on one side, cropped by the
  frame, for a dynamic full-bleed composition."*
